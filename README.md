# NAM Web App - 로그인 페이지

깔끔한 로그인 폼 디자인입니다.

## 기능

- 이메일/비밀번호 로그인
- 로그인 유지 옵션
- 아이디·비밀번호 찾기 링크
- 소셜 로그인 (카카오톡, 네이버, 페이스북, 구글)
- 회원가입 링크

## 사용 방법

VS Code Live Server나 다른 정적 파일 서버로 `index.html`을 열어주세요.

## 파일 구조

```
.
├── index.html      # 로그인 페이지
├── styles.css      # 스타일시트
├── app.js          # JavaScript
└── images/
    └── logo.png    # 로고 이미지
```
